# Templates

A collection of starter templates with pre-configured tooling and architecture patterns.

## Available Templates

| Template | Stack | Description |
|---|---|---|
| [react-router-template](./react-router-template/) | React 19 + Vite + Keycloak | Web app with auth, routing, and state management |

## Usage

1. Copy the template folder into a new project directory
2. Read the template's `README.md` for setup instructions
3. Configure environment variables based on the provided `.env.local` example
4. Run `npm install` then `npm run dev`
